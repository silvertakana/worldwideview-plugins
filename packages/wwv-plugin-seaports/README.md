# @worldwideview/wwv-plugin-seaports

WorldWideView plugin — **Seaports**.

## Data Source

- **Source:** OpenStreetMap via [Overpass Turbo](https://overpass-turbo.eu/)
- **Tag:** `harbour=yes`
- **Format:** Static GeoJSON (4975 features)
- **License:** [ODbL](https://opendatacommons.org/licenses/odbl/) (OpenStreetMap)

## Installation

```bash
npm install @worldwideview/wwv-plugin-seaports
```

## Peer Dependencies

- `@worldwideview/wwv-plugin-sdk`
