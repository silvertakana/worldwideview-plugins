# @worldwideview/wwv-plugin-lighthouses

WorldWideView plugin — **Lighthouses**.

## Data Source

- **Source:** OpenStreetMap via [Overpass Turbo](https://overpass-turbo.eu/)
- **Tag:** `man_made=lighthouse`
- **Format:** Static GeoJSON (14653 features)
- **License:** [ODbL](https://opendatacommons.org/licenses/odbl/) (OpenStreetMap)

## Installation

```bash
npm install @worldwideview/wwv-plugin-lighthouses
```

## Peer Dependencies
