# @worldwideview/wwv-plugin-airports

WorldWideView plugin — **Airports**.

## Data Source

- **Source:** OpenStreetMap via [Overpass Turbo](https://overpass-turbo.eu/)
- **Tag:** `aeroway=aerodrome`
- **Format:** Static GeoJSON (46074 features)
- **License:** [ODbL](https://opendatacommons.org/licenses/odbl/) (OpenStreetMap)

## Installation

```bash
npm install @worldwideview/wwv-plugin-airports
```

## Peer Dependencies

- `@worldwideview/wwv-plugin-sdk`
